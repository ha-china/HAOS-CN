## Home Assistant OS 极速版 🌟
- 🚀 为中国地区专门加速的 **Home Assistant OS**  

- 🔧 不修改任何系统内的其它信息，所有的修改都以加速为目的。 

- ⚠️ **禁止未经授权并超过10个客户端上进行公司商用**

- 🛠️ 每个月**27日**系统由 **GitHub Actions** 自动构建。
